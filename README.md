Une visualisation interactive de l'histoire de l'Open Source avec un thème inspiré de GitLab pour la Nuit de l'Info 2025.

## Démarrage rapide

1. **Installation des dépendances**
   ```bash
   npm install
   ```

2. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```
